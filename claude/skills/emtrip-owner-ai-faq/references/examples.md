# FAQ実データサンプル（池原酒造・高橋商店）

これはSupabaseのfaq_masterに実際に登録されているデータです。

## 重要：q_normの品質基準

| 悪い例（修正前） | 良い例（修正後） | 理由 |
|---|---|---|
| 駐車場 あり | 駐車場 あり 瀬長島ウミカジテラス 駐車可 車 停められる | 固有名詞なし・語数不足 |
| 感染症対策 なし | 感染症対策 コロナ 消毒 マスク 特別な対策なし シーサー注意 | 2語では機能しない |
| 写真撮影 OK 可 | 写真撮影 OK 可 店内 インスタ SNS 撮影可能 | 3語では機能しない |
| 商品カスタマイズ 不可 なし | カスタマイズ 不可 オーダーメイド 名入れ 特注 シーサー注意 | 汎用語のみ |
