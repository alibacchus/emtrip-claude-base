# EmTrip 法的チェックスキル

## 0. 前提

- ハルシネーション厳禁。法的判断の最終確認は弁護士に委ねる
- このスキルは「危険箇所の特定」と「確認すべき論点の整理」が目的
- hourei MCPが使える場合は条文を必ず参照してから判断する
- 推論・仮説は必ず明記する

---

## 1. EmTripに関係する主要法令

### 景品表示法（優良誤認・有利誤認）
**EmTripで特に注意すべきポイント：**
- 「中川社長と話せる」表現 → 本人がリアルタイムで返信していると誤認させないこと
- 「初月無料」の表記 → 自動有料移行の有無・条件を申込前に明示すること
- 無料期間の表示 → 「登録から1か月間無料」（初月無料より正確）
- 打消し表示は主コピーの近くに読める大きさで配置すること

**許容できる表現：**
- 「中川社長とAIで話せる」＋「中川社長の言葉・価値観・経験をもとに、AIが対話をつなぎます」

**避けるべき表現：**
- 「中川社長本人が返信しています」
- 「中川社長に直接つながります」

### 個人情報保護法
**EmTripで取得する情報：**
- LINE USER ID（識別情報）
- 登録日時・利用ログ
- 会話データ
- 課金情報（Stripeが管理）

**チェック項目：**
- 利用目的の特定・明示（プライバシー通知に記載）
- 委託先（Stripe・Supabase・Vercel・LINE）への安全管理義務
- 不要になったデータの削除・利用停止対応
- 会話データをAI学習に使わないことの明記

### 消費者契約法
**EmTripで注意すべきポイント：**
- 不当条項の排除（消費者に一方的に不利な条項は無効）
- 管轄裁判所：消費者に著しく不利な管轄は無効になる場合がある
  → 東京地方裁判所（合意管轄）が現在の設計
- 解約・返金条件の明示

### 特定商取引法
**EmTripの課金設計に関係する条項：**
- 定期購入（月額サブスク）の表示義務
  - 料金：月額980円（税込）✅
  - 更新サイクル：月次自動更新 ✅
  - 解約方法：LINE解約メニューまたはinfo@emtrip.io ✅
  - 解約の効果：次回更新日前日まで利用可 ✅
- 申込前の最終確認画面での必須表示項目

## 2. チェックフロー

### Step 1：対象の特定
何をチェックするか明確にする
- コピー・広告文言か
- 契約書・利用規約か
- プロダクトの機能・導線か

### Step 2：関連法令の特定
| 対象 | 主な関連法令 |
|------|------------|
| 広告・POP・コピー | 景表法 |
| 利用規約・契約書 | 消費者契約法・特定商取引法 |
| 個人情報取扱い | 個人情報保護法 |
| 課金・サブスク | 特定商取引法・消費者庁ガイドライン |
| 副業・業務委託 | 労働基準法・下請法 |

### Step 3：hourei MCPで条文確認
```
# hourei MCPの使い方
「景品表示法第○条を参照して」
「個人情報保護法の利用目的の特定に関する条文は？」
「消費者契約法の不当条項規定を確認して」
```

### Step 4：リスク分類
| レベル | 判定基準 | 対応 |
|--------|---------|------|
| 高 | そのままだと法令違反の可能性 | 即修正必須 |
| 中 | 解釈次第でリスクあり | 修正推奨または説明準備 |
| 低 | 交渉論点・印象の問題 | 判断はアリーに委ねる |

### Step 5：修正文案の提示
リスクがある場合は修正前・修正後を並べて提示する。

## 3. EmTrip固有の確認済み事項

### 解消済みリスク
- OpenAI API：デフォルトでモデル学習不使用。「汎用AIモデルの学習目的に提供しない」の表現は問題なし
- Stripe委託：課金情報はStripeが管理。プライバシー通知への委託先明記済み
- 自動課金なし：無料→有料は自動移行なし。有料登録後の自動更新は別途明示済み

### 要注意事項（継続確認）
- Stripe `cancel_at_period_end: true` の実装確認（有料ユーザーが出る前に）
- 蔵開き専用QR・通常QRの2系統実装確認（宮城さんへ）
- 弁護士レビュー：文書A優先（消費者法専門推奨）

## 4. 副業・業務委託に関するチェック

EmTripの石垣・宮城との業務委託契約で注意すべき点：

- 労働者性の判断（指揮命令・時間拘束・場所の制約があると労働者と判断されるリスク）
- 源泉徴収の要否（個人への報酬は10.21%源泉徴収が必要）
- 社会保険の適用（月次定額委託は雇用と見なされるリスク）
- 下請法の適用（資本金1,000万円以下のEmTripから個人への委託は対象外だが確認を）

- 金融商品取引法：投資家への説明資料で「確実に儲かる」等の断定的な表現は禁止
- エンジェル投資：適格機関投資家等特例業務の要件確認
- EDINET：上場後の開示義務（今は関係なし）
