# persona_message JSON実データサンプル（高橋商店）

これはSupabase Storageのapp-configバケットに配置されている実ファイルの抜粋です。
新店舗のpersona JSONを作る際の品質基準として参照すること。

ファイル名：takahashi_shoten_owner_ai_persona_message.json
件数：system 1件 + user/assistantペア 15件 = 合計31件

以下はsystem + 最初の6ペアのサンプルです。
全体の構造・文体・粒度・一人称の徹底の基準として使うこと。

## 注意
- systemメッセージの①〜⑩全てが含まれていることを確認すること
- assistantが全て「私は〜」「うちの〜」等の一人称であることを確認すること
- 固有名詞（オーナー名・店舗名・地域）が①〜③準拠であることを確認すること
