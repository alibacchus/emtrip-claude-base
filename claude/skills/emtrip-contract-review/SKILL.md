## 2. 3層文書設計
文書A：実証実験基本契約書（B2B・EmTrip↔高橋商店）
文書B：運用ルールブック（現場運用・EmTrip担当↔高橋商店関係者）
文書C：エンドユーザー向け利用条件・プライバシー通知（B2C）

### 文書間整合チェックリスト
1. サービス名：全文書でココロクロス
2. 実証期間：2026年4月25日〜7月31日で統一
3. ユーザー区分：区分A（4/25〜5/31・1か月）・区分B（6/1〜7/31・24時間）が全文書で一致
4. 削除タイミング：個別30日猶予（B・C）と全体終了後90日（A）の役割分離が明記
5. 自動課金なし：全文書で明記
6. 管轄裁判所：全文書で東京地方裁判所（合意管轄）統一
7. 問い合わせ：一次LINE・二次info@emtrip.io・高橋商店は案内のみで統一
8. 停止時保護：区分A・区分B・有償の3パターンが明記

## 3. B側・C側が構えるポイント

### B（高橋商店）
- 運用負荷・責任分担を連想させる表現
- 「共同提供」など責任感を与える言葉
- → 「高橋商店との実証実験としてご提供しています」が適切

### C（エンドユーザー）
- 自動課金・自動削除の誤解を招く表現
- 区分の差の説明が不明確な表現

### 高橋商店からの想定質問と回答根拠
| 質問 | 回答根拠 |
|------|---------|
| 料金告知タイミング | C重要ポイント・第3条 |
| 自動削除か継続確認か | B継続フロー・C第3条 |
| 月額課金タイミング | C第4条 |
| 問い合わせ対応 | A第11条・B第4章・C第9条 |
| 途中停止時の有償ユーザー | A第13条・B第5-1 |
| 実証期間後 | A第13条・C第7条 |
| 高橋商店の利用料 | A第5条「無償」 |
