**2026-04-30**：OPePeの位置づけを「最小工数PSF検証」から「サイドビジネスとして勝ちに行く」へ転換。最終的にはEmTripメイン事業（ココロクロス／emtrip-miniapp／emtrip-webchat）と統合する前提でインフラ・スキル・思想を揃える。

**2026-05-08（追加）**：v3.6戦略レポートでココロクロス2チャネル並列運用に重心が移ったが、OPePeは並走継続。**石垣島のダイビングショップにて約1ヶ月のPSF検証**を実施する方針。優先順位は依然４位だが、PSF検証中ゆえ「最小工数」原則は緩めて運用可。

**Why:**
- ココロクロス3店舗のうち池原酒造・シーサー注意も沖縄県内 → 同タイミングで石垣島ダイビング店PSF検証を走らせるのは地理的・時間的に効率良い
- v3.6 §41 5月-7月の最優先タスク（決算／公庫／SNS／Web Phase α-ε）と直接競合せず、石垣秀樹の沖縄ネットワーク経由でリーチ可能
- 「マリン事業者向け当日事故防止」のコアドメインに最も近い検証対象がダイビングショップ

**How to apply:**
- OPePeに関する判断は「最小工数」を絶対条件としない（PSF検証で必要な範囲は許容）。ただしまこと工数を要する実装は7月まで原則NG（CLAUDE.mdルール）
- 優先順位はココロクロス3店舗→決算→マツリクロス→OPePeを維持
- 石垣島ダイビング店PSFの観察項目：当日事故防止オペレーション・参加者ブリーフィング・遅刻場所不明返却フロー・天候判断のうち、どれが現場で最も痛みがあるか
- PSF検証期間は約1ヶ月（推論：2026-05-08着手→6月上旬の週次レビュー対象）。期間終了時に「ココロクロスPSFと比較して投資配分をどう変えるか」を月次経営判断に乗せる
- emtrip-webchat / OPePe / emtrip-miniapp の3プロジェクトは将来統合される前提（project_three_repos_integration.md 参照）
- OPePeのコーポレートカラーは未確定 → 暫定でEmTripカラー（#4653A2 / #ED6D46）継承、最終統合時に再検討
- 「OPePeに引っ張られてココロクロスがズレた」が観測された瞬間に止める（CEOアドバイザー監視ルール）
