EmTrip 公式 MVV（2026-05-08更新）：

| 項目 | 公式定義 |
|---|---|
| **Mission** | みんなの思い出をみんなの価値に変える |
| **Vision** | 個人の思い出を社会の資産へ循環させる |
| **Concept** | AI×現地体験×デジタル技術で「体験=データ=IP=価値」を実装する |
| **タグライン** | 思い出を価値に変える会社 |
| **事業定義** | 地域住民・職人・青年会の一次ナラティブをAIでインターフェース化し、旅前・旅中・旅後の対話から文化の意味データを蓄積する文化共創SaaS |

Why: 過去にClaudeがMissionを「思い出を価値に変える」と単数形で誤記した経緯あり。正確には「**みんなの思い出をみんなの価値に変える**」（双方向の集合性が本質）。タグラインは短縮形で「思い出を価値に変える会社」。

How to apply: 公式文書・ピッチ・契約書では Mission のフルバージョン「みんなの思い出をみんなの価値に変える」を使う。短い場面ではタグライン「思い出を価値に変える会社」でも可。
