**ベロ族**：蔵元・社長個人のストーリーや哲学への深い興味は薄く、酒と場の楽しみを目的に来た消費型の参加層。

Why: 2026年4月の髙橋商店酒蔵開きPoC（配布440枚・純粋アクティブ0.25%）で多数観察された母集団の主成分。これにより EmTrip の元仮説「店主への伝声管」（深いファン化前提）が構造的に反証された。過去にClaudeが「ベロ族 = 深い興味層」と逆に解釈した経緯あり。語感に騙されない。

How to apply:
- ベロ族 ≠ ファン層／深い興味層。**消費型・カジュアル層**を指す
- ベロ族の存在は「ファン深耕仮説の母集団薄さ」を示すデータポイント
- 訴求軸転換の根拠：ベロ族にも価値が即届くのは「観光客の制約解除（4ペイン解消）」であり、伝声管的なストーリー深耕ではない
- ピッチ・チーム会話・ドキュメント全てでこの定義を維持
- 関連定義：4ペイン = ①言語の壁・②情報の非対称・③意思決定の負荷・④アクセス
